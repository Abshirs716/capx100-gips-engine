Sheet 3 Cumulative Calculation

üéØ Purpose

Shows the step-by-step compounding calculation that produces the cumulative return. This proves we use geometric (not arithmetic) compounding as required by GIPS 2020.

Column Definitions

Column Header Description
D Return Monthly return percentage
E 1+R Return expressed as multiplication factor (e.g., +2.10% = 1.0210)
F Product Running cumulative product
G Calculation Shows the actual multiplication: "1.021 √ó 1.0185"
H Value Dollar value of $100 investment

üìê The Compounding Formula

Cumulative Return = ‚àè(1 + Ri) - 1

This means: Multiply all the (1 + monthly returns) together, then subtract 1

Step by step:
Month 1: 1.0000 √ó 1.0210 = 1.0210
Month 2: 1.0210 √ó 1.0185 = 1.0399
Month 3: 1.0399 √ó 0.9680 = 1.0066
...
Month 60: ... = 1.8242

Final: 1.8242 - 1 = 0.8242 = 82.42%

Sheet 7 Drawdown Analysis

üéØ Purpose

Tracks peak-to-trough declines throughout the investment period. Shows when drawdowns occurred, how severe they were, and recovery status. Critical for understanding downside risk.

Column Definitions

Column Header Description
D Return Monthly return (color coded)
E Value Current portfolio value
F Peak Highest value achieved so far
G Drawdown % below peak (negative when in drawdown)
H Status "Peak" or "In DD"

üìê Drawdown Formula

Drawdown = (Current Value - Peak) / Peak

If Value = Peak, Drawdown = 0% (at new high)
If Value < Peak, Drawdown = negative (in drawdown)

Maximum Drawdown = -10.60%
Occurred during Q3 2022 (bear market)
